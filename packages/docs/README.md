# StreamLine Docs

Professional documentation site powered by [Fumadocs](https://fumadocs.dev) + Next.js.

## Develop

From the monorepo root:

```bash
npm install
npm run dev:docs
```

Open [http://localhost:3001](http://localhost:3001) — docs live at `/docs`.

## Build

```bash
npm run build:docs
```

## Content

MDX sources: [`content/docs`](./content/docs).

Engineering plans (not the public site) remain in the repo-root [`docs/`](../../docs/) folder.
